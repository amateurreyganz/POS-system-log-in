# 🛒 Ganzon's Smart POS System

A modern Point of Sale (POS) System built using **HTML**, **CSS**, and **JavaScript**. This project provides a simple and user-friendly interface for managing products, calculating sales, and processing customer transactions.

---

## 📌 Features

- 🔐 Login Authentication
- 🏠 Dashboard
- 🛒 Shopping Cart
- ➕ Add Products to Cart
- ➖ Increase/Decrease Quantity
- ❌ Remove Products
- 💰 Automatic Sales Calculation
- 📈 Profit Calculation
- 🧾 VAT (12%) Computation
- 💵 Cash Tendered & Change Calculation
- 🔍 Product Search
- 🚪 Logout System
- 📱 Responsive Design

## 🚀 How to Run

1. Download or clone this project.
2. Open the project folder in Visual Studio Code.
3. Make sure all files are inside their correct folders.
4. Open **index.html** using Live Server.
5. Log in using the provided credentials.
6. Start adding products and perform checkout.
